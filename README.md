# Homebrew Tap for Rustyman

This is the official Homebrew tap for [Rustyman](https://github.com/johnsonlee/rustyman), a high-performance MITM HTTP/HTTPS proxy written in Rust.

## Installation

```bash
brew tap johnsonlee/rustyman
brew install rustyman
```

## Usage

```bash
rustyman --help
```

## Updating

```bash
brew update
brew upgrade rustyman
```
